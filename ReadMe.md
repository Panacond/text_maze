# you can create your maze
it's example:
```
g-+ +-s
  | | |
x-+-+-+
```
## designations
- g - gate
- `-` - horisontal way
- | - vertical way
- `+` - crossroads
- s - gamer (start position, put at the crossroads)
- x - treasures (put at the crossroads)
- ' ' - space is space place
